# ClaudeRouelleBot
Spits out random quotes daily from ClaudeRouelle

# Requirements:
- node.js

# Installation
- Make directory for the bot: mk NAME_OF_DIRECTORY
- set up json file and use defaults: npm init
- Install discord.js: npm install discord.js --save
- copy:
  . index.js
  . quotes.txt
  to to the directory.
